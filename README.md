# PocketFetch

A self-hosted personal media downloader built with FastAPI.

## Features

- Analyze supported URLs
- Download media
- Queue downloads
- Live progress
- Installable PWA

## Tech Stack

- FastAPI
- yt-dlp
- HTML
- CSS
- JavaScript
- Docker
